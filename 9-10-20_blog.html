This week I researched the mix-blend-mode property in CSS. Mix-blend-mode is used to blend elements with the background or with other elements. Most of the uses of this property are for displaying text over a background image/video or having an image/video display through the text. I found this to be very interesting as it can give a webpage a more unique look by stacking text and images while still maintaining that each can be seen. Many modern websites make use of this property, especially travel sites to show video/images while still providing links and text about their services. An example of this is the visit Virginia Beach site, playing a video behind the text on the homepage.

Visit Virginia Beach

The mix-blend-mode CSS tag is used to achieve this effect and can be set to blend either text through an image or image through text. Setting mix-blend-mode: screen allows an image to show text. An image with text on top is done by setting the text color to white with a black background. There are many other variations to this tag and ways to use this property and I hope to use this property in future projects.

Example CSS code for displaying text with an image displaying under it: .backdrop { width: 600px; height: 210px; background-image: url(someimage.jpg); background-size: 100%; margin: auto; } .text { color: black; background-color: white; mix-blend-mode: screen; width: 100%; height: 100%; font-size: 160pt; font-weight: bolder; text-align: center; line-height: 210px; margin: 0; }

Example CSS code for displaying an image with text over it: .text { color: white; background-color: black; mix-blend-mode: screen; width: 100%; height: 100%; font-size: 160pt; font-weight: bolder; text-align: center; line-height: 210px; margin: 0; }
