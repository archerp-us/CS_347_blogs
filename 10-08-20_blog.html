While JavaScript is used for most animations, CSS also can control some aspects of animation and design. This week I found one way that CSS can be used to create animations is through scroll snapping. This feature is useful for stopping the scroll to show a specific section of text or an image. Many websites make use of this feature to display their products in individual sections. An example of this can be on Teslaâ€™s website here.

Scroll snapping is mainly useful for ensuring that information on a webpage will be viewed in the best possible manner by visitors and customers. The use of CSS for these type of animation features makes the code more cohesive since code can just be added to the style section instead of writing in JavaScript. Scroll snapping is accomplished with the scroll-snap-type and scroll-snap-align features in CSS, which control the position of the container. Overall this feature is useful for adding some style to the scroll aspect of a website and is especially good for advertising products.
